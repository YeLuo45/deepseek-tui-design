# DeepSeek TUI 架构概览

## 项目简介

DeepSeek TUI 是一个基于 Rust 构建的终端编码 Agent，使用 DeepSeek V4 系列模型（deepseek-v4-pro / deepseek-v4-flash）。它提供 1M Token 上下文窗口、流式推理块显示、完整工具集（文件操作、Shell 执行、Git、Web 搜索等）。

## 技术栈

| 层级 | 技术选型 |
|------|----------|
| 编程语言 | Rust |
| TUI 框架 | ratatui |
| HTTP 客户端 | reqwest |
| 序列化 | serde |
| 状态管理 | tokio + parking_lot |

## 模块组织

### Entry Point

- **`main.rs`** - CLI 参数解析（clap）、配置加载、入口路由

### Core Components

| 模块 | 路径 | 说明 |
|------|------|------|
| **core** | `crates/core/` | 主引擎组件，Agent 循环、会话管理、事件系统 |
| **agent** | `crates/agent/` | Agent 逻辑、工具调用、状态机 |
| **tui** | `crates/tui/` | 终端用户界面、布局、输入处理 |
| **tui-core** | `crates/tui-core/` | TUI 核心组件（复用层） |

### Tool Layer

| 模块 | 路径 | 说明 |
|------|------|------|
| **tools** | `crates/tools/` | 工具注册表、Shell、文件操作、Git |
| **execpolicy** | `crates/execpolicy/` | 执行策略/审批流程 |
| **hooks** | `crates/hooks/` | 前后置钩子扩展点 |
| **mcp** | `crates/mcp/` | Model Context Protocol 客户端 |

### Infrastructure

| 模块 | 路径 | 说明 |
|------|------|------|
| **config** | `crates/config/` | TOML 配置解析、默认值 |
| **secrets** | `crates/secrets/` | API Key 安全存储 |
| **state** | `crates/state/` | 会话状态持久化 |
| **protocol** | `crates/protocol/` | LLM 输入输出协议处理 |

### External Interfaces

| 模块 | 路径 | 说明 |
|------|------|------|
| **cli** | `crates/cli/` | 命令行参数定义 |
| **app-server** | `crates/app-server/` | HTTP/SSE 运行时 API 服务器 |

## 三种运行模式

| 模式 | 说明 | 使用场景 |
|------|------|----------|
| **Plan** | 只读探索模式，不执行危险操作 | 代码审查、架构探索 |
| **Agent** | 交互式模式，每个操作需要用户审批 | 日常开发、精确控制 |
| **YOLO** | 自动审批所有操作 | 快速原型、简单任务 |

## 关键特性

- **Auto Mode** - 自动选择模型和思考级别
- **Thinking-mode Streaming** - 实时显示推理过程
- **1M Token Context** - 超长上下文支持
- **Session Save/Resume** - 会话保存与恢复
- **Workspace Rollback** - 工作区快照与回滚
- **Durable Task Queue** - 持久化任务队列
- **HTTP/SSE Runtime API** - 无头 Agent 工作流
- **LSP Diagnostics** - 编辑器诊断集成
